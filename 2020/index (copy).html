Visão geral da Internet das Coisas (IoT)


O que é?

IoT é um paradigma inovador que está rapidamente ganhando terreno no cenário das telecomunicações sem fio moderna [Atori, Iera, Morabito, 2012].

Visão geral da Internet das Coisas (IoT)


A ideia básica é a presença generalizada em torno de nós de uma variedade de coisas ou objetos - como tags de identificação por radiofrequência (RFID), sensores, atuadores, telefones celulares, etc. Os quais podem interagir uns com os outros e cooperar com seus vizinhos para alcançar metas comuns [Atori, Iera, Morabito, 2012]


Em outras palavras:

É uma rede de objetos físicos que possuem tecnologia embarcada com sensores e atuadores capazes de coletar e transmitir dados.

Quais atuais características tecnológicas favorecem o uso de IoT?

  • Baixo custo dos dispositivos embarcados;

    • Redução do consumo de energia;

    • Baixo custo dos sensores;

  • Aumento da conectividade dos dispositivos;

  • Uso de api externas (IBM platform - Bluemix/Google/Amazon);

  • Produção de muitos dados que podem ser explorados;

  • Aumento da capacidade das baterias e redução de tamanho.

O que é o DIGITU?


Um projeto que surgiu para resolver o problema de autenticação em portas.

Principais componentes:



App1: Aplicativo capaz de mandar mensagens para controlar as portas (IONIC e MQTT);

S1: Servidor responsável pela comunicação arduino e o SCEFA (MQTT);

SCRIPT: código embarcado para arduino;

SCEFA: Software web para controle de acesso e frequência com autenticação biométrica (DJANGO).



Objetivos gerais


1.Desenvolver um sistema de autenticação biométrica genérico e de baixo custo.


2.Suportar vários contextos (sistema de ponto, frequência, sala inteligente).


3.Ser conectado, moderno, flexível e competitivo.

Quais os requisitos técnicos atendidos?



Uso de tecnologias eficazes, escaláveis e simples.


Python, Django;


Arduino, NodeMcu e Raspberry;


MQTT - protocolo de redes usado em IoT;


Tecnologias conhecidas, seguras, poderosas, genéricas e flexíveis.


Quais os requisitos estudantis atingidos?



Uso de tecnologias acessíveis aos nossos alunos;


Amplo campo de pesquisa em mecatrônica e informática;


Prováveis colaborações com prefeituras e IFs.



Quais os requisitos institucionais contemplados?



Solução para automatizada para frequência de alunos;


Solução para controle de acesso em portas;
