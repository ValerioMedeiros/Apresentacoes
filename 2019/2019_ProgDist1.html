BTestBox





BTestBox é um gerador automático de teste para linguagem B. Ele gera testes que atendem critérios de cobertura de código de aplicações crítica (aviação, trens)




Conceitos (1/3)





Tempo de execução:

Ts: tempo de execução serial,

Tp: tempo de execução paralela.


speedup is defined as the ratio of the serial runtime (Ts) of the best sequential algorithm for solving a problem to the time taken (Tp) by the parallel algorithm to solve the same problem on p processors.

speedup = S = Ts / Tp

Fonte: Jun Zhang - Department of Computer Science. University of Kentucky.

Conceitos (2/3)





Eficiência

is defined as the ratio of speedup (S) to the number of processors (p). Efficiency measures the fraction of time for which a processor is usefully utilized.





Eficiência = E = S / p = (Ts / p*Tp)

Fonte: Jun Zhang - Department of Computer Science. University of Kentucky.

Conceitos (3/3)





Escalabilidade

is a measure of a parallel system’s capacity to increase speedup in proportion to the number of processors.





Fonte: Jun Zhang - Department of Computer Science. University of Kentucky.
